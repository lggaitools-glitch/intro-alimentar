# IntroAlimentar — Forum Presence Playbook v1

## Goal
Build trust-driven acquisition for IntroAlimentar through Reddit first, using a personal-ish, human tone.

This is **not** a spam/link-drop strategy.

Success looks like:
- parents recognizing the account as helpful
- recurring qualified traffic to blog + food database
- questions that reveal product gaps
- future email capture / Instagram growth

---

## Core Principles

1. **Help first, link second**
   - 80% answers with no link
   - 15% soft mentions
   - 5% direct links only when clearly relevant

2. **Personal-ish, not fake**
   - Write like a thoughtful human
   - Do not pretend to be a parent if that’s not true
   - Do not invent personal experiences

3. **Evidence-based, not preachy**
   - Mention AEP / OMS when useful
   - Avoid medical absolutes
   - Recommend pediatrician follow-up when needed

4. **Stay useful in English-speaking subs**
   - Most Reddit opportunities are in English
   - If relevant, frame IntroAlimentar as a Spanish-language resource for Spanish-speaking families

5. **Protect the account**
   - No bursts of repetitive comments
   - No self-promo in early days
   - No copy-paste replies

## Priority Communities (Reddit First)

### Tier 1 — Highest intent

1. **r/BabyLedWeaning**
   - Best fit for BLW, food prep, readiness, gagging vs choking, first foods
   - High intent, topic-specific
   - Strong moderation likely; avoid direct promo early

2. **r/NewParents**
   - Broad but highly relevant
   - Good for readiness signs, first foods, routines, allergen questions
   - Better for soft educational replies

3. **r/beyondthebump**
   - Large and active
   - Better for relatable, practical answers than hard technical posts
   - Good for “what worked / what matters” style replies

### Tier 2 — Secondary opportunities

4. **r/toddlers**
   - Useful later for picky eating / food transitions
5. **r/Parenting**
   - Broader, lower intent, but occasional fit
6. **r/breastfeeding**
   - Only when complementary feeding intersects with milk/formula timing

### Tier 3 — Spanish-language / region-specific exploration

7. Spanish parenting subreddits if active enough
8. Spain-focused communities only when a post is directly relevant

## Account Positioning

### Tone
- calm
- practical
- non-judgmental
- concise
- evidence-aware

### Voice pattern
Use this shape:
1. validate the concern
2. answer the actual question
3. give 2–4 practical points
4. add one caveat if needed
5. optional soft mention only if genuinely helpful

Example:
> If your baby is around 6 months and can sit with minimal support, has good head control, and isn’t pushing food straight out with the tongue, those are the real signs I’d look for. Interest in your plate alone isn’t enough. I’d prioritize iron-rich foods first and keep textures appropriate for whichever approach you use.

## Link Strategy

### Phase A — warm-up (first 10–14 days)
- no direct links unless explicitly requested
- focus on helpful comments
- target: 2–4 replies/day
- goal: account credibility + community feel

### Phase B — soft mentions
- mention resource without forcing link
- example:
  - “I’ve been organizing this into a Spanish food database for parents because the same questions keep coming up.”
- only send link if someone asks

### Phase C — selective direct linking
Direct links allowed only when:
- the person explicitly asks for a resource
- the thread is asking for a guide/checklist/database
- the community norm clearly tolerates it

Preferred links:
1. blog article
2. food database page
3. homepage only if nothing more specific exists

---

## Posting Rhythm

### Weekly cadence
- **15–20 comments/week**
- **2 original posts/week max**
- **0–2 links/week** in the beginning

### Daily workflow (20–30 min)
1. Scan target subreddits for fresh posts (<12h)
2. Pick 3–5 threads with real relevance
3. Reply to 2–3
4. Save 2 more for later if needed
5. Log what got traction

## First 20 Reply Angles

1. Readiness signs matter more than just age
2. Interest in food is not enough by itself
3. Mixed approach is valid; doesn’t have to be BLW or purées only
4. Iron matters early — prioritize iron-rich first foods
5. Gagging is common; choking is different
6. Soft foods can still be served in BLW-safe shapes
7. Allergens shouldn’t be delayed without reason
8. Start with one new food at a time when anxious
9. Repeated exposure matters more than one successful try
10. Milk/formula remains primary at the start
11. Don’t force quantities in the first weeks
12. Mealtime posture matters a lot
13. Avoid honey, whole nuts, excess salt, excess sugar
14. Egg can be introduced early when well cooked
15. Yogurt can be offered before cow’s milk as a main drink
16. Constipation after starting solids is common but manageable
17. Mess is learning, not failure
18. Appetite varies wildly day to day
19. You don’t need fancy baby recipes to start
20. Family foods adapted safely usually work best

## Soft CTA Patterns

Use only when natural.

### No-link soft mention
- “I’ve been organizing this into a Spanish resource because these questions repeat a lot.”
- “I have a food-by-food version of this in Spanish if that would help.”

### Link-if-asked pattern
- “If you want, I can share the exact checklist / food page I use.”

### Direct link pattern
- “This guide is the clearest version I’ve put together on this topic: [link]”

---

## What to Never Do

- don’t post the same comment repeatedly
- don’t argue with parents choosing purées over BLW
- don’t claim medical authority
- don’t force the brand into unrelated threads
- don’t drop homepage links when a specific page is better
- don’t overuse “according to science” language
